# Healthful — Ingredient Safety Analyzer

Is your product really safe? Paste any ingredient list, product name, or URL — we check every ingredient against EU, FDA, WHO, IARC & more.

## Features

- **Global Database Checks**: EU, FDA, WHO, IARC, EWG, CDSCO, Prop 65.
- **Dynamic Scoring**: 1-10 safety score with detailed risk metrics.
- **Categorized Results**: Flagged vs. Safe ingredients.
- **Full Reports**: Detailed summary stats and recommendations.

## Local Setup

To run locally:
1. Clone the repository.
2. Open `index.html` in your browser.
3. Or use a local server: `npm start`.
